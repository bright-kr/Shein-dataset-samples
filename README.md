# Shein 데이터셋 샘플

<h2>1001개 Shein 제품의 샘플 データセット</h2>

![Shein dataset header](https://github.com/bright-kr/Shein-dataset-samples/blob/main/shein-datasets.PNG)

1000개 이상의 제품으로 구성된 Shein データセット 샘플입니다. データセット는 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>이 무료 データセット에 포함된 데이터 포인트:</h2>

* ```product_name```: 제품의 이름 또는 제목입니다
* ```description```: 제품에 대한 텍스트 설명입니다
* ```initial_price```: 제품의 원래 또는 시작 가격입니다
* ```final_price```: 할인 또는 프로모션 적용 후의 현재 또는 최종 가격입니다
* ```currency```: 가격이 표시되는 통화입니다
* ```in_stock```: 제품의 현재 재고 여부를 나타냅니다(True/False)
* ```color```: 제품에 제공되는 색상입니다
* ```size```: 제품에 제공되는 사이즈입니다
* ```reviews_count```: 고객이 남긴 리뷰 또는 평점의 수입니다
* ```main_image```: 제품을 대표하는 메인 이미지입니다
* ```category_url```: 제품 카테고리와 연관된 URL 또는 링크입니다
* ```url```: 제품 페이지로의 URL 또는 링크입니다
* ```category_tree```: 제품이 속한 카테고리의 계층 트리 구조입니다
* ```country_code```: 판매 국가 또는 원산지를 나타내는 국가 코드입니다
* ```domain```: 제품이 등록된 도메인 또는 웹사이트입니다
* ```image_count```: 제품과 연관된 전체 이미지 수입니다
* ```image_urls```: 제품과 관련된 이미지로 연결되는 URL입니다
* ```model_number```: 제품과 연관된 모델 번호(SKU)입니다
* ```offers```: 제품과 연관된 특별 혜택 또는 딜 정보입니다
* ```other_attributes```: 제품의 추가 속성 또는 특징입니다
* ```product_id```: 제품과 연관된 고유 식별자 또는 코드입니다
* ```rating```: 고객이 부여한 제품의 평균 평점입니다
* ```related_products```: 현재 제품과 관련된 다른 제품 정보입니다
* ```root_category```: 제품이 속한 루트 또는 최상위 카테고리입니다
* ```top_reviews```: 제품의 상위 또는 추천 리뷰입니다
* ```category```: 제품이 속한 특정 카테고리입니다
* ```brand```: 제품과 연관된 브랜드 또는 브랜드명입니다
* ```all_available_sizes```: 각 제품에 대해 사용 가능한 모든 사이즈 목록입니다

그리고 훨씬 더 많은 항목이 있습니다.

이는 "Shein Products (public data)" データセット에서 파생된 샘플 하위 집합이며,
해당 データセット에는 <b>32,800,000개 이상의 제품</b>이 포함되어 있습니다.

사용 가능한 データセット 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, or Parquet. Optionally, files can be compressed to .gz</b>.

データセット 전송 유형 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 주기: <b>Once, Daily, Weekly, Monthly, Quarterly, or Custom basis</b>.

추출된 데이터 포인트에 추가로 제공되는 데이터 보강: <b>요청 기반입니다.</b>

<b>[전체 Shein データセット 받기](https://brightdata.co.kr/products/datasets/shein)</b>.

<h2>Shein データセット의 활용 사례는 무엇입니까?</h2>

<h3>1. 브랜드 감성 분석</h3>
제품 리뷰와 평점을 심층적으로 분석하여 소비자 의견을 파악하고, 귀사의 제공 상품이 시장 기대치에 부합하도록 확인할 수 있습니다. Shein データセット를 활용하여 특정 제품 또는 브랜드 전반에 대한 고객 감성을 이해함으로써 상업 전략을 고도화하는 데 도움이 됩니다.

<h3>2. 소비자 수요 인사이트</h3>
재고 공백을 포착하고, 특정 제품에 대한 증가하는 수요를 탐지하며, 소비자 트렌드를 식별할 수 있습니다. Shein データセット는 기업이 재고 관리에서 전략적 의사결정을 내리고, 재고 수준을 최적화하며, 공급망을 보다 효율적으로 간소화할 수 있도록 지원합니다.

<h3>3. 경쟁력 있는 가격 전략</h3>
경쟁사 제품군 내 유사 제품과 카테고리를 식별하여 견고한 가격 전략을 수립할 수 있습니다. Shein データセット를 활용해 최적 가격을 결정하고, 가격 격차를 발견하며, 실시간 시장 데이터를 기반으로 동적 가격 모델을 구현할 수 있습니다.
